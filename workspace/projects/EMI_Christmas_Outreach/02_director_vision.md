# 導演願景與鏡頭序列 (Director's Vision & Shot List)

**視覺風格:** 「從霜凍到火焰 (From Frost to Fire)」。高對比度。
*   **世界 (The World):** 冷色調 (青色、板岩藍)，高銳利度，動態模糊，混亂的構圖。
*   **聖所 (The Sanctuary):** 暖色調 (琥珀色、金色)，柔和擴散，穩定的構圖，慢動作。

**鏡頭序列 (Shot List Breakdown):**

| Clip ID | 時長 | 蒙太奇描述 (Montage Description) |
| :--- | :--- | :--- |
| **Clip_01** | 8s | **[0s-3s]** 廣角鏡頭：Alex 在繁忙、下雪的城市街道上逆風行走，背景是模糊的冷色霓虹燈。 <br> **[3s-6s]** 特寫：他的腳步沉重地踏在雪泥中，手緊緊抓著沉重的皮包帶子。 <br> **[6s-8s]** 中景：他停下來靠在牆上，呼出的白氣在冷空氣中消散，眼神疲憊。 |
| **Clip_02** | 8s | **[0s-3s]** 過肩鏡頭：Alex 的視線穿過人群，聚焦在街道對面 EMI 教會溫暖發光的入口。 <br> **[3s-5s]** 主觀鏡頭 (POV)：模糊的車流中，教會的十字架或標誌清晰可見，彷彿在召喚。 <br> **[5s-8s]** 特寫：Alex 的臉部，藍色的反光中映出一點金色的光芒，表情從麻木轉為渴望。 |

| **Clip_04** | 8s | **[0s-4s]** 極特寫：Alex 的眼睛閉著，淚水滑落，臉上洋溢著金色的暖光。 <br> **[4s-8s]** 廣角/拉遠：Alex 坐在長椅上，身影在巨大的彩色玻璃窗或十字架光輝下顯得渺小但被保護，字幕浮現。 |
